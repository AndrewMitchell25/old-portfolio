Capture the Flag

2017

I created this Capture The Flag game using Unity in 2017, during my freshman year of high school. It's based off of a prototype I made using Scratch when I was in 7th grade, which was one of my first experiences with programming. Building games like this one using Unity was how I developed most of my basic programming skills. I even turned my game into a mobile app on my phone.
