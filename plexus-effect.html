Plexus Effect

2018

Plexus Effect is a project I created using Unity and C#. It's a cool design created by spawning points that gradually disappear as they fall down the screen and drawing lines between them if they are close enough to each other. At the time, I was still fairly new to scripting, and this project helped me learn about arrays and using loops to run through them to affect each object in the array.
